Project

Because using the Groundspeak Wherigo emulator on my Apple computer was a pain, I decided to create an online version of this application, allowing me to run cartridges from my favorite web browser!

The webwigo project was born!

Started as a pet project, it grows to the point that I decided to release it to the community with the hope that it can be useful to authors of Wherigo cartridges.

Wherigo Foundation

The emulator itself is part of a global project named the Wherigo Foundation. Read the announcement in the Geocaching forum (free registration required) to get more information and/or if you want to know how to participate!
